# 📊 E-Commerce Performance Dashboard

A Power BI dashboard built to monitor business performance for a wellness 
e-commerce brand selling health and wellness products.

> ⚠️ Concept Project — built using simulated data aligned with a real business use case.

## 🔗 Live Dashboard
[View Published Dashboard](https://app.powerbi.com/links/y_1rxZlH_3?ctid=ff28a236-881c-4b29-9b77-71405b66aca2&pbi_source=linkShare)

## 📌 Business Problem
A wellness e-commerce company needed visibility into:
- Sales and revenue trends
- Customer behavior and retention
- Product-level performance
- Marketing and funnel efficiency

## 🎯 Project Objectives
- Monitor monthly business performance
- Support cross-functional decision-making
- Ensure accurate and consistent reporting

## 📈 Key Metrics
| Metric | Value |
|---|---|
| Total Revenue | €459.65 |
| Total Orders | 10 |
| Average Order Value (AOV) | €45.97 |
| Gross Margin % | 65% |

## 🔍 Key Insights

### Business Performance
- 70% of orders from new customers, 30% returning
- Paid Search and Paid Social are top acquisition channels

### Funnel & Conversion
- Total Sessions: 9,730
- Add to Cart: 803
- Checkouts: 550
- Orders: 415
- Strong drop-off between sessions and add-to-cart — optimization opportunity

## 🛠️ Technologies Used
- Power BI / Looker
- Google Sheets (data source)
- Data Modeling
- Basic Forecasting
